# Fraud Transaction Detection Project

A Machine Learning based Fraud Transaction Detection System built using Flask and Random Forest Classifier.

## Features
- Fraud prediction system
- Transaction history
- Flask web interface
- Machine learning model
- SQLite database integration

## Technologies Used
- Python
- Flask
- Scikit-learn
- HTML/CSS
- SQLite

## Installation

```bash
git clone https://github.com/upendher5G1/Fraud-Transaction-Detection-Project.git
cd Fraud-Transaction-Detection-Project
pip install -r requirements.txt
python app.py
```

## Project Structure

- `app.py` → Main Flask application
- `train.py` → Model training script
- `fraud_model.pkl` → Trained ML model
- `templates/` → HTML pages
